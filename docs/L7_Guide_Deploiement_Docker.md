# Livrable 7 : Guide de Déploiement Docker

## 1. Prérequis
Pour installer et lancer la solution HealthAI Coach, vous devez disposer des outils suivants :
* **Docker** (version 20.10+)
* **Docker Compose** (version 2.0+)
* **Git** (pour le clonage du dépôt)

## 2. Procédure de mise en œuvre (Installation en < 30 min)
Suivez ces étapes pour déployer l'environnement complet :

1. **Clonage du projet** :
   ```bash
   git clone [https://github.com/Kabouri-Anass/HealthAI-Coach.git](https://github.com/Kabouri-Anass/HealthAI-Coach.git)
   cd HealthAI-Coach
